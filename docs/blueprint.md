# **App Name**: Turfista

## Core Features:

- Turf Discovery Home: An inviting home page showcasing available sports turfs through interactive cards, detailing image, name, location, sport type, hourly price, and user ratings.
- Intelligent Search & Filtering: Robust search capabilities to find turfs by name, and dynamic filters for sport type (Cricket/Football) and specific areas within Mysuru.
- Comprehensive Turf Detail Pages: Dedicated pages offering a deep dive into each turf with multiple high-resolution images, detailed descriptions, an embedded Google Maps location, opening hours, and direct contact information.
- Instant WhatsApp Booking Integration: Streamlined 'Book on WhatsApp' functionality present on both turf listing cards and detail pages, facilitating quick reservation inquiries.
- Admin Content Management: An administrative panel to efficiently add new turf listings, modify existing turf details, delete outdated entries, and upload associated turf images (using Firestore for data and Firebase Storage for assets).
- AI-Enhanced Turf Descriptions: An admin tool utilizing generative AI to automatically craft compelling and engaging marketing descriptions for turfs based on provided characteristics and amenities, streamlining content creation.

## Style Guidelines:

- Primary accent color: Vibrant Neon Green (#1AFF73), selected for its energetic and modern sports feel, to highlight interactive elements and call-to-actions.
- Background color: A deep, almost black shade (#141714), subtly infused with the primary green hue, providing a premium dark-themed canvas.
- Secondary accent color: A warm, light yellow-green (#AFDE75) for complementary details and secondary highlights, creating visual depth and contrast against the dark background.
- Headlines and prominent text will use 'Space Grotesk' (sans-serif) for its modern, slightly technical and bold appearance, fitting the premium sports theme.
- Body text and functional UI elements will use 'Inter' (sans-serif), chosen for its exceptional readability across various screen sizes and its clean, contemporary aesthetic, complementing the headlines.
- Clean, minimalistic line icons with subtle fills, adopting the primary neon green for emphasis to align with the modern sports UI and enhance visual clarity against the dark theme.
- Mobile-first responsive design, employing a structured card-based layout with generously rounded corners for turfs and data. Emphasis on clear spatial separation for a premium feel.
- Smooth, subtle transition animations and hover effects, particularly for turf cards and interactive elements, to enhance user engagement and provide a polished, dynamic experience.
